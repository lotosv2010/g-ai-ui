# 组件列表

当前组件体系参照 Ant Design X 的核心交互模型（`Welcome / Conversations / Prompts / Attachments / Bubble / Sender`），用于快速搭建 AI 对话产品的主流程页面。

## 核心组件

- [Welcome](./welcome.mdx)：欢迎区与场景引导
- [Conversations](./conversations.mdx)：会话列表与上下文切换
- [Prompts](./prompts.mdx)：快捷提示词入口
- [Attachments](./attachments.mdx)：附件状态展示
- [Bubble](./bubble.mdx)：消息渲染单元
- [Sender](./sender.mdx)：输入发送器
