# 组件列表

`@g-ai-ui/ui` 参考 Ant Design X 的会话式交互范式，围绕 AI 产品主路径提供六个核心组件。

## 核心组件

- [Welcome](./welcome.mdx)：首屏欢迎区与任务引导
- [Conversations](./conversations.mdx)：会话列表与上下文切换
- [Prompts](./prompts.mdx)：快捷提示词推荐
- [Attachments](./attachments.mdx)：附件上传状态管理
- [Bubble](./bubble.mdx)：消息渲染单元
- [Sender](./sender.mdx)：输入与发送控制
