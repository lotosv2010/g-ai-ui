# 组件列表

组件层聚焦 AI 对话场景的核心交互原语，强调“可组合、可扩展、低耦合”。

- [ChatBubble 对话气泡](./chat-bubble.mdx)：负责消息展示与角色语义表达
- [ChatInput 对话输入](./chat-input.mdx)：负责输入采集与发送行为触发
